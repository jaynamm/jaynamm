⚙︎ **시스템 엔지니어** 로 기상청 슈퍼컴퓨터의 서버, 네트워크, 스토리지 등의 시스템을 다루어보면서 리눅스를 잘 다루게 되었고  
⚙︎ **인프라 엔지니어** 로 인프라를 운영하면서 서버, 네트워크, 보안 등 여러 이슈를 경험하고 운영한 경험이 있고
